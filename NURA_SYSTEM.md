# NÜRA — Sistema de comandos

## Jerarquía de responsabilidades

| Comando | Responsabilidad | Puede tocar | NO puede tocar |
|---------|----------------|-------------|----------------|
| NÜRA EVOLVE | Producto, UX, conversión, retención | Funcionalidades, flujos, onboarding, estructura | Detalles visuales sin impacto en UX |
| NÜRA DISEÑO | Visual, branding, tipografía, jerarquía, premium | CSS, componentes visuales, organización | Lógica de negocio, funcionalidades |
| NÜRA FIX | Bugs, rendimiento, estabilidad, deuda técnica | Correcciones de código, responsive, accesibilidad | Nuevas funcionalidades, rediseños |

## Regla de oro
Ningún comando deshace mejoras de otro sin razón objetiva demostrable.
La evolución es acumulativa. Cada iteración hace Nüra mejor que antes. Nunca simplemente diferente.

## Prioridad ante conflicto
1. Experiencia del usuario
2. Confianza
3. Utilidad real
4. Producto
5. Diseño
6. Optimización técnica
